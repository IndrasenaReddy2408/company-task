Html HTML (HyperText Markup Language) is the standard markup language used to create and structure web content, defining elements like headings, paragraphs, images, and links. It acts as the skeleton for web pages, using tags to annotate text and organize content for web browsers to interpret.
Css CSS (Cascading Style Sheets) is the standard language used to style and lay out web pages. While HTML provides the structure (like the skeleton of a building), CSS provides the presentation (like the paint, lighting, and decor).
JavaScript JavaScript is a powerful, lightweight, and dynamically typed programming language primarily used to create interactive and dynamic content for websites. Alongside HTML and CSS, it is one of the three core technologies of the World Wide Web.
Java Java is a high-level, class-based, object-oriented programming (OOP) language originally developed by James Gosling at Sun Microsystems in 1995 and now owned by Oracle. It is famous for its "Write Once, Run Anywhere" (WORA) philosophy, meaning compiled Java code can run on all platforms that support Java without the need for recompilation.
